1. 안드로이드에서 PWA 설치
Chrome 브라우저에서 웹사이트 열기:

배포한 웹사이트 URL을 Chrome 브라우저에서 엽니다.
홈 화면에 추가:

브라우저 메뉴 (우측 상단의 점 3개)에서 "홈 화면에 추가"를 선택합니다.
나타나는 팝업에서 "추가" 버튼을 누릅니다.
앱처럼 사용:

홈 화면에 아이콘이 추가되며, 이를 클릭하면 웹사이트가 네이티브 앱처럼 전체 화면 모드로 실행됩니다.
2. iOS에서 PWA 설치
Safari 브라우저에서 웹사이트 열기:

배포한 웹사이트 URL을 Safari 브라우저에서 엽니다.
홈 화면에 추가:

하단의 공유 버튼 (네모 안에 화살표 아이콘)을 누릅니다.
스크롤하여 "홈 화면에 추가" 옵션을 선택합니다.
나타나는 화면에서 "추가" 버튼을 누릅니다.
앱처럼 사용:

홈 화면에 아이콘이 추가되며, 이를 클릭하면 웹사이트가 네이티브 앱처럼 전체 화면 모드로 실행됩니다.
